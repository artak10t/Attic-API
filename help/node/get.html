Node Get

REQUIRED PRIVILEGE - NONE

Get node settings.

Request

'POST/GET' methods are accepted

RESPONSE

Returns after request

Error Code Message Data Description
0 Success All node information

EXAMPLE

{
  "code": 0,
  "msg": "",
  "data": {
    "fqdn": "attic.com",
    "registration_mode": 0,
    "session_timeout": 1440,
    "activation_invitation_tokens_timeout": 4320,
    "password_email_tokens_timeout": 60,
    "account_expire_delete_timeout": 14400,
    "max_space": 10737418240,
    "max_attics_count": 10,
    "max_folders_count": 10000,
    "max_files_count": 100000000,
    "max_depth": 100
  }
}
