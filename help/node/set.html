Node Edit Admin

REQUIRED PRIVILEGE - ADMIN

Set node settings.

Request

'POST' methods are accepted


TAKES

Field Mandatory Default Type Constraints Description
session_id Yes No Default string session_id
fqdn Yes No Default string Max size 255 Fully qualified domain name
registration_mode Yes No Default int 0 (Private), 1 (Invitation), 2 (Public) Registration Mode
sessions_timeout Yes No Default int Has to be > 0 Sessions timeout in minutes
activation_invitation_tokens_timeout Yes No Default int Has to be > 0 Activation and invitation tokens timeout in minutes
password_email_tokens_timeout Yes No Default int Has to be > 0 Password change and email change tokens timeout in minutes
account_expire_delete_timeout Yes No Default int Has to be > 0 Unactivated and requested to delete accounts tokens timeout in minutes
max_space Yes No Default int Has to be >= 0 Maximum allowed space for accounts in bytes
max_attics_count Yes No Default int Has to be > 0 Maximum allowed attics count for accounts
max_folders_count Yes No Default int Has to be > 0 Maximum allowed folders count for accounts
max_files_count Yes No Default int Has to be > 0 Maximum allowed files count for accounts
max_depth Yes No Default int Has to be > 0 Maximum allowed folders depth for accounts

EXAMPLE

{
  "session_id": "763c10b83cfd69da3f416ac2f9b0d11a535cc177922de4827a01b76080714133",
  "fqdn": "attic.com",
  "registration_mode": "0",
  "sessions_timeout": "1440",
  "activation_invitation_tokens_timeout": "4320",
  "password_email_tokens_timeout": "60",
  "account_expire_delete_timeout": "14400",
  "max_space": "10737418240",
  "max_attics_count": "10",
  "max_folders_count": "10000",
  "max_files_count": "100000000",
  "max_depth": "100"
}

RESPONSE

Returns after request

Error Code Message Data Description
0 Success
-1 E_NOT_LOGGED_IN Invalid session
-3 E_UNAUTHORIZED Account is not an admin
-4 E_DISABLED Account is disabled
-5 E_ACTIVATED Account is not activated
-50 E_ONLY_POST Only POST method allowed
-51 E_FIELD_NOT_SET One or more mandatory fields are not set
-52 E_FIELD_INVALID One of the fiels is not valid

EXAMPLE

{
  "code": 0,
  "msg": "",
  "data": {}
}
