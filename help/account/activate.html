Account Activate

REQUIRED PRIVILEGE - NONE

Uses activation_token to activate the associated account.

Request

Only 'POST' method is accepted


TAKES

Field Mandatory Default Type Constraints Description
token Yes No Default string Activation token

EXAMPLE

{
  "token": "942A5659AD83851E54BC8D818764BE9D126F25E510EC9C233AE57F25D757BBD4"
}

RESPONSE

Returns after request

Error Code Message Data Description
0 Success
-9 E_INVALID_TOKEN Invalid token
-50 E_ONLY_POST Only POST method allowed
-51 E_FIELD_NOT_SET One or more mandatory fields are not set

EXAMPLE

{
  "code": 0,
  "msg": "",
  "data": {}
}
