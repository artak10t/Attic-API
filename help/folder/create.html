Folder Create

REQUIRED PRIVILEGE - USER/ADMIN

Create new folder.

Request

Only 'POST' method is accepted


TAKES

Field Mandatory Default Type Constraints Description
session_id Yes No Default string Session id
name Yes No Default string No space, size 1-255 char Folder name
attic_id No No Default string Whenever creating foreign folder this field is mandatory. Id of the attic where new folder will be created
parent_id No No Default string null (If root) Whenever creating foreign folder this field is mandatory. Id of the parent folder where new folder will be created. Set null if want to create inside attic
account_id No No Default string ADMIN Whenever creating foreign folder this field is mandatory. Id of the account where new folder will be created

EXAMPLE

{
  "session_id": "942A5659AD83851E54BC8D818764BE9D126F25E510EC9C233AE57F25D757BBD4",
  "name": "folder1"
  "attic_id": 2,
  parent_id: null,
  account_id: 15
}

RESPONSE

Returns after request

Error Code Message Data Description
0 Success session_id Session token
-1 E_NOT_LOGGED_IN Invalid session
-3 E_UNAUTHORIZED Account is not a admin
-3 E_UNAUTHORIZED Foreign account has higher privileges
-4 E_DISABLED Account is disabled
-5 E_ACTIVATED Account is not activated
-6 E_DOESNT_EXIST Foreign account doesn't exist
-6 E_DOESNT_EXIST Attic doesn't exist
-6 E_DOESNT_EXIST Attic doesn't belong to given account
-6 E_DOESNT_EXIST Parent folder doesn't exist
-11 E_MAX_COUNT Max folder depth reached
-50 E_ONLY_POST Only POST method allowed
-51 E_FIELD_NOT_SET One or more mandatory fields are not set
-52 E_FIELD_INVALID One of the fiels is not valid

EXAMPLE

{
  "code": 0,
  "msg": "",
  "data": {}
}
