Account Set

REQUIRED PRIVILEGE - USER/ADMIN

Set attic settings.

Request

Only 'POST' method is accepted


TAKES

Field Mandatory Default Type Constraints Description
session_id Yes No Default string Session id
attic_id Yes No Default int Attic id
name Yes No Default string No space, size 1-64 char Attic unique name within account
access Yes No Default int 0 (Private), 1(Authorized), 2(Public) Attic access, changing this also changes files access
description Yes "" String Size 1-500 Attic description
enabled Yes if attic_id is from foreign account ADMIN, int 1 (true), 0 (false) Is attic enabled

EXAMPLE

{
  "session_id": "942A5659AD83851E54BC8D818764BE9D126F25E510EC9C233AE57F25D757BBD4",
  "attic_id": 16,
  "name": "Attic1",
  "access": 2,
  "name": "Sample attic"
}

RESPONSE

Returns after request

Error Code Message Data Description
0 Success session_id Session token
-1 E_NOT_LOGGED_IN Invalid session
-3 E_UNAUTHORIZED Account is not a moderator/admin
-3 E_UNAUTHORIZED Foreign account has higher privileges
-4 E_DISABLED Account is disabled
-5 E_ACTIVATED Account is not activated
-6 E_DOESNT_EXIST Attic doesn't exist
-50 E_ONLY_POST Only POST method allowed
-51 E_FIELD_NOT_SET One or more mandatory fields are not set
-52 E_FIELD_INVALID One of the fiels is not valid

EXAMPLE

{
  "code": 0,
  "msg": "",
  "data": {}
}
